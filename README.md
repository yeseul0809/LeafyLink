## 1. 사용한 라이브러리

### tanstack-query

서버 상태 관리를 관리 하기 위해 사용하였으며, 비동기 로직을 간편하게 작성하기 위해 사용했어요

### supabase

Supabase를 백엔드로 사용하여 실시간 데이터베이스를 사용했어요

### tailwind css

재사용성이 높고, 코드의 가독성과 유지보수성을 향상시킬 수 있기에 사용했어요
